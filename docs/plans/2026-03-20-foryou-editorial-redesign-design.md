# For You Editorial Redesign Design

## 背景

浏览器插件 side panel 的 `For You` 区块已经具备推荐头部、池子状态和推荐列表三层结构，但当前视觉重心仍然偏散：

- 头部卡与推荐卡的视觉体量过于接近，内容主角感不够
- 状态 chips 更像系统面板，而不是内容入口的一部分
- 推荐卡内部标题、推荐理由、操作区的层级不够清晰，扫读效率一般

这次优化不改数据协议，也不改交互链路，只提升推荐页首屏的审美和阅读节奏。

## 目标

1. 让第一屏更像内容推荐入口，而不是运营信息面板
2. 把推荐卡的视觉主角明确收敛到封面和标题
3. 保留 `换一批`、候选池摘要和现有反馈交互，不引入新的行为复杂度
4. 在窄宽 side panel 内维持更稳的层级、留白和节奏

## 方案

采用“编辑式内容流”方向：

### 1. 头部降噪

- `For You` 保留，但从视觉锚点降成更轻的导语标识
- 标题改成更明确的两行层级：一句主标题 + 一句更短的说明
- `换一批` 继续作为首要操作，但从“控制按钮”气质调整为更像内容流中的轻操作
- 三枚状态 chips 保留，但压缩成更轻的摘要行，避免与推荐卡争抢体量

### 2. 推荐卡重排

- 整卡改成更干净的上下信息节奏：封面与主题标签在上，标题和推荐理由在中，操作区在下
- 标题字号、行高和字重更坚定，提升扫读优先级
- 推荐理由收短并弱化装饰，作为“为什么值得点开”的一层辅助说明
- 元信息与状态标签统一成更细的字级，减少视觉噪音

### 3. 视觉语气

- 延续现有亮色、轻内容产品方向，不做更重的玻璃或过深的科技感
- 背景层次从“多处都有渐变”改成“重点区域有、普通区域收敛”
- 用更克制的描边、阴影和圆角节奏，减少甜腻感，往内容编辑感靠拢

## 实现边界

本次允许调整：

- `extension/popup/popup.html` 中推荐视图的头部和卡片相关 markup
- 同文件内对应 CSS 布局、配色、阴影、间距和响应式规则
- 相关布局测试断言
- 插件文档与 changelog

本次不改：

- `popup.js` / `popup-helpers.js` 的推荐数据协议
- 推荐反馈、换一批、续页和 runtime-stream 逻辑
- 画像与聊天 tab

## 验证口径

自动化需覆盖：

1. 推荐头部仍保留 `For You`、标题、说明、`换一批` 和三枚状态 chips
2. 推荐卡改为更明确的纵向信息分区，而不是旧的平均密度结构
3. 窄宽场景下头部与卡片不会退化成拥挤堆叠

手动期望：

1. 第一张推荐卡进入首屏后更像主内容
2. 头部信息仍然能快速扫到，但不再喧宾夺主
3. 整体气质更克制、更耐看，不像临时拼出来的组件板
