# Popup UI Refresh Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 将浏览器插件 popup 重构为更明亮、统一、贴近 Bilibili 风格的内容面板，同时保持推荐、画像、聊天三类现有功能不变。

**Architecture:** 继续复用现有 popup 业务逻辑和 API 调用，只重做 popup 的 HTML 结构、CSS 视觉系统和少量 DOM 渲染细节。通过先补 helper 测试、再改结构和样式、最后更新文档与验证的顺序控制回归风险。

**Tech Stack:** Chrome Extension MV3, vanilla HTML/CSS/JavaScript, Node test runner, existing extension build pipeline
